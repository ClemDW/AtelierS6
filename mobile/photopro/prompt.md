## Prompt 2 — Constitution

```
je veux créer une app mobile pour ios et android destinée aux photographes
professionnels pour partager leurs galeries photos avec leurs clients.
```

---

## Prompt 3 — Specification

```
Le client doit pouvoir voir les galeries publics quand il arrive sur l'application,
il doit pouvoir accéder à une galerie privé en entrant le lien ou le code d'accés de
la galerie, quand le client doit pouvoir afficher une galerie. Le photographe doit
pouvoir se connecter à l'application et doit pouvoir ajouter une photo à son stock photo.
```

---

## Prompt 4 — Plan

```
On utilisera flutter pour faire l'application mobile et on utilisera l'api dans le dossier
AtelierS6 qui est deux dossiers en arrière. Pour l'affichage galerie on utilisera la gateway
front et pour l'upload et la connexion photographe on utilisera la gateway back.
```
